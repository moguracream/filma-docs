    動画の公開

    ← クイックスタートへ戻る

    画面URL

    • ファイル編集: /filmaadmin/file/edit/:file_id
    • ファイル一覧: /filmaadmin/file

    単体公開

    1. ファイル一覧から対象の「編集」を開く
    2. 「公開する」をON
    3. 必要に応じて「公開期限」を設定（空欄は期限なし）
    4. 「保存」

    一括公開/非公開

    1. ファイル一覧で対象ファイルにチェック
    2. 「選択したファイルの公開状態」で「公開」または「非公開」を選択
    3. 「一括変更」

    絞り込みと確認

    • 一覧上部で「公開状態」「タグ」「カテゴリ」でフィルタ可能
    • ソートとページングにも対応

    公開期限について

    • 表示可否: 公開期限（published_until）を過ぎると、公開フラグがONでも「外部公開の対象外」になります。
      • APIの一覧取得では除外されます。
      • 埋め込み/プレイヤーへのアクセスは不可（404相当の応答）になります。
    • 管理画面表示: 一覧の「公開状態」で「公開期限切れ」と表示され、詳細/編集画面では公開期限に過去日時が表示されます。
    • 期限なし: 公開期限を空欄にすると期限なし（常に公開）として扱われます。
    • 期限の再設定: 未来の日時を再設定すれば、再び公開対象として扱われます。

    • 期限付きURLとの関係: 期限付きURLは各URLに設定した期限で制御されます（公開期限とは独立）。

    • APIのshow_allパラメータについて:

      • デフォルトではAPIの一覧は「公開中かつ期限内」のみ返します。
      • ただし、show_all=true を付与し、かつAPIキーがfullaccessのときは「非公開や期限切れも含む全件」を取得できます。
      • また、管理者（filmaadmin）ログイン状態でのAPIアクセスも同等に全件表示が有効になります。
      • 例: /filmaapi/storage?show_all=true

# 動画の非公開化・削除

## 非公開化

- 個別: `/filmaadmin/file/edit/:file_id` → 「公開する」をOFF → 保存
- 一括: `/filmaadmin/file` で対象にチェック → 「選択したファイルの公開状態」= 非公開 → 「一括変更」

## 削除

1. `/filmaadmin/file/detail/:file_id` → 「削除」
2. 確認画面で内容を確認 → 削除を実行
3. 一覧に成功メッセージ表示

## 削除不可ケース

- 作成者が「プラットフォーム管理者」のファイルは削除不可（エラーメッセージが表示されます）

# メディア/動画の登録・管理

会員機能で使う動画の用意は、既存の「動画のアップロード」「動画の公開」の手順に従います。本章では会員向けに意識すべきポイントだけをまとめます。

## 手順の概要

1. フォルダを用意: 必要に応じて `/filmaadmin/folder` でフォルダを作成。
2. アップロード: `/filmaadmin/file` からファイルをアップロード（詳細: [動画のアップロード](upload.md)）。
3. 公開設定: ファイル編集画面で「公開する」をONにし、必要なら公開期限を設定（詳細: [動画の公開](publish.md)）。
4. Mediafile確認: 「派生メディア」カード（クリックで開閉）で生成されたMediafileを確認。SKU紐づけに利用します。

## 会員機能での注意点

- 公開状態: SKUに紐づけるMediafileは「公開」状態にしてください。非公開のままでは再生できません。
- 解像度ごとのMediafile: 同一の視聴プランに複数解像度を紐づける場合、目的のMediafileをSKUに割り当てます。
- 期限付きURL: 会員機能では通常不要です（会員JWTで再生するため）。社外共有など特別なケースのみ発行します。
- アイテムスコープ再生: 視聴プラン（ITEM）スコープでは「親」側のMediafile（配信マニフェスト）を再生対象とします。エンコードで生成された「子」側のMediafileはSKU（配信種別/解像度）経由で使う前提です。

## トラブルシュート

- 再生できない場合は「公開状態」「期限切れ」「対応解像度のMediafileが有効か」を確認してください。
- ファイルが見つからない場合は、アップロード先フォルダと公開設定を再確認してください。

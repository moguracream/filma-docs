# 会員機能の設定

単一ショップ運用を前提としたショップ情報や問い合わせ先の設定をまとめます。

## ショップ情報の編集

- 画面: 管理画面の「ショップ設定」(`/filmaadmin/shop`)
- 設定項目: ショップ名、ロゴ、問い合わせ先、表示用リンクなど。
- 公開/非公開: 必要に応じて公開状態を切り替えます。非公開にすると会員向け画面での表示/遷移に影響するため、再生確認も合わせて実施してください。

## 会員画面での表示要素

- ショップ名/ロゴ/問い合わせ先は会員画面のヘッダやフッタで参照されます。更新後は会員サイト側で表示を確認してください。

## 注意点

- 単一ショップ運用のため追加の紐づけ操作は不要です。
- 問い合わせ先メールが未設定の場合、会員からの問い合わせ導線が不明確になるため必ず設定してください。

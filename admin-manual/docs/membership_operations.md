# 運用・トラブルシュート

会員機能の運用で頻出する確認ポイントと対処方法をまとめます。

## よくあるエラーと対処

- 認証コードが届かない: メールアドレス誤入力、迷惑メール、連続リクエストの間隔を確認。
- 認証エラー/期限切れ: 再ログインまたはページ再読込を試す。
- 視聴権エラー: 対象会員に正しいSKU/視聴プラン視聴権が付与されているか、期限切れでないか確認。
- 再生不可: Mediafileが公開状態か、視聴プランが非公開になっていないか、紐づけ漏れがないかを確認。

## 公開設定と紐づけチェックリスト

- Mediafile: 公開 ON / 期限内 / 対象解像度が有効
- 視聴プラン: 公開 ON
- SKU: 対象Mediafileが紐づいている
- 視聴権: SKU/視聴プランが正しく付与され、期限切れではない
- 表示挙動: SKU権限の場合はSKUに紐づく動画（ダウンロード配信も含む）、視聴プラン/ショップ権限の場合は親Mediafileのストリーミングのみ表示される
- 会員: 有効状態でログインできる

## ログ確認ポイント

- 管理画面/会員ログイン: 認証コード送信、JWT発行失敗時のログを確認。
- 再生系: JWT検証エラー、視聴権エラー時のレスポンスやサーバーログを確認。
- 期限付きURL（特例使用時）: 発行・アクセス回数のログを確認し、不要になったURLは無効化する。

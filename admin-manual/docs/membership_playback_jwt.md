# 再生と認可のポイント

会員も管理者も、ログイン後は再生に必要な権限が自動で付与されます。仕組みの詳細を意識せずに運用できるよう、確認ポイントだけをまとめます。

## 基本

- ログイン状態で再生リンクを開けば、必要な認可は自動で付与されます。
- 権限や期限は自動更新されるため、再生途中で途切れることは通常ありません。

## トラブルシュート

- 「認証に失敗」「有効期限切れ」などで再生できない場合: 再ログインまたはページ再読込を試してください。
- 「視聴権がありません」等のメッセージ: 会員への視聴権付与、視聴プラン/Mediafileの公開設定を確認してください。
- `item_disabled` / `sku_disabled` エラー: 視聴プランやSKUが無効の場合に返ります。視聴プランを有効にし、必要なSKUが有効になっているか確認してください（視聴プランを無効にすると配下SKUも無効扱いになります）。SHOP/ITEM/SKU のスコープに応じて付与範囲を見直してください。

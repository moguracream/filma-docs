# 会員機能の概要

会員登録/ログイン、視聴プラン・視聴権の管理、再生確認までをまとめたガイドです。ショップは1件のみ運用し、全ての会員・視聴プラン・視聴権がこのショップに紐づく前提です。

## 対象読者

- Filma 管理画面にログインできる管理者（メール認証コードでログイン）
- 会員登録フローのサポート、視聴プラン・視聴権の設定、再生確認を担当する方

## 前提条件

- 管理画面にログインできること
- 動画のアップロード/公開方法は既存マニュアル（[動画のアップロード](upload.md), [動画の公開](publish.md)）を参照済み

## よく使う場所とURL（会員機能）

- 管理画面:
  - 会員一覧（会員管理）: `/filmaadmin/customer`
  - 視聴プラン一覧: `/filmaadmin/item`
  - ショップ設定（会員機能設定）: `/filmaadmin/shop`
- 会員向け画面:
  - ログイン: `/filmacustomer/login?shop_id=...`
  - 閲覧可能な視聴プラン: `/filmacustomer/items?shop_id=...`
  - マイページ: `/filmacustomer/profile`
  - 退会: `/filmacustomer/withdraw`

## クイックスタート（最短で再生確認する手順）

1. **動画を用意する**: ファイルをアップロードし公開する（[動画のアップロード](upload.md) / [動画の公開](publish.md)）。
2. **視聴プラン/SKUを作る**: 管理画面の視聴プランメニューで視聴プランを作成し、SKUにMediafileを紐づける（[視聴プラン・SKU管理](membership_item_sku.md)）。
3. **会員を用意する**: 会員サイトから自登録するか、管理者が追加して有効化する（[会員管理](membership_customers.md)）。
4. **視聴権を付与する**: 管理画面の会員詳細から対象会員へ SKU/視聴プラン 単位で権限を付与（[視聴権管理](membership_entitlements.md)）。
5. **再生確認する**: 会員サイトの「閲覧可能な視聴プラン」から再生リンクを開き、再生できることを確認（[運用・トラブルシュート](membership_operations.md)）。

## セッションのポイント

- 管理画面へログイン後、再生や管理操作に必要な権限は自動で付与されます。
- 会員サイトでもログイン後は再生に必要な権限が自動で付与されます。再生できない場合は再ログインと権限設定を確認してください。

次の章「管理者・権限の基本」で、単一ショップ運用における権限モデルとログイン/セキュリティの注意点を説明します。

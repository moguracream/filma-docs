# ユーザーについて（一般ユーザーとAPIユーザー）

## 一般ユーザー

- 役割: 管理画面での運用（アップロード/公開/期限付きURL発行など）
- 要件: メール必須（重複チェックあり）

## APIユーザー

- 役割: 外部サイト/システム連携のためのAPI利用
- 特徴: APIキー保有、API種類（`readonly`/`fullaccess`）、アクセス許可ドメインで制限
- 作成/変更: 一般→APIへ変更時にAPIキー自動発行。API設定は別画面で編集
- 注意: `fullaccess` は強力。キーの保護とローテーションを徹底

## デフォルトのAPIユーザー

- 初期アカウント発行時に自動作成されます
- 埋め込みHTML でもこのユーザーの「アクセス許可ドメイン」設定が必要です
- 設定場所: `/filmaadmin/apisettings/edit/:user_id`
